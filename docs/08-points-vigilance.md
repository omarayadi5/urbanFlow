# Points de vigilance

- Ne jamais versionner `.env`.
- Remplacer le secret JWT exemple avant tout lancement.
- En production, activer `COOKIE_SECURE=true`.
- Configurer `ALLOWED_ORIGINS` avec l'URL Vercel exacte.
- Render ne fournit pas MySQL par defaut : prevoir une base externe compatible.
- Les donnees transport et routing sont demo tant que les APIs officielles ne sont pas integrees.
- Le livrable de soutenance doit justifier l'usage de l'IA et citer les sources.
